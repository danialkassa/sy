# Welcome to Ningbo Siyang CMS

This guide helps you manage the website content. Here are the most common tasks:

## How to Add a New Product
1. Go to **Products** in the sidebar
2. Click **Add Product**
3. Fill in the **required fields** (marked with *): Product Name, SKU, Category, Main Photo
4. Add a description and specifications as needed
5. Click **Save** to save as draft, or **Publish** to make it live

## How to Edit the Homepage
1. Go to **Pages** → **Homepage**
2. Scroll to the section you want to change (Hero, Stats, Featured Products, etc.)
3. Edit the text, images, or settings
4. Click **Save** to save as draft

## How to Add a Blog Post
1. Go to **Blog** in the sidebar
2. Click **Add Blog Post**
3. Fill in: Title, Slug (URL-friendly name), Author, Category, Excerpt
4. Write your content in the Markdown editor
5. Add a featured image
6. Click **Save** to save as draft

## How to Change Company Info
1. Go to **Settings** → **Company**
2. Edit: Phone, Email, Address, Business Hours, Social Media links
3. Click **Save** — changes go live after publishing

## How to Edit Navigation
1. Go to **Settings** → **Navigation**
2. Edit main menu links or product category links
3. Click **Save**

## Understanding the Workflow
- **Draft**: Your changes are saved but NOT visible on the website
- **In Review**: Your changes are ready for someone to approve
- **Published**: Your changes are LIVE on the website

## Tips
- Always preview your changes before publishing
- Use the "Regenerate Index" button if search results seem outdated
- The sync status badge (bottom-right) shows if the website is up to date
- If sync fails, click the red badge to retry
